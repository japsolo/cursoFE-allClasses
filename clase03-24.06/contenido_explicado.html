Formularios en HTML

Los formularios en HTML nos sirven para captar información. Escenarios de uso más comunes:

  • Contacto
  • Buscador
  • Login
  • Registro

Etiqueta <form>

Le da funcionalidad al formulario. Hace que el mismo funcione. Tiene como mínimo 2 atributos que son obligatorios:

  • method=""
    Este atributo permite setear como se maneja la información del formulario. Sólo existen dos métodos: get y post.
  • action=""
    Este artributo nos permite setear una ruta en la cual se va a procesar la información del formulario. Generalmente en el action, se define una ruta que trabaja la información desde el backend.

Etiqueta <input>

Sirve para generar varios campos de información (es multifunción). La etiqueta input, necesita de 2 atributos obligatorios:

  • type=""
    Define que tipo de campo queremos crear. Existen varios valores posibles, ellos son:
    • text: define un campo de inserción de texto.
    • password: define un campo de tipo contraseña.
    • radio: define un campo de tipo radio-button o también llamado botón de opción.
    • checkbox: define un campo de tipo selección o también llamado botón de checkbox.
    • file: define un campo para subir archivos.
  • name=""
    Define un nombre para ese campo.

Etiqueta <label>

Nos permite generar un texto que viene a servir cómo RÓTULO del campo del formulario.

Etiqueta <textarea>

Esta etiqueta sirve para generar un campos de múltiples líneas de texto.

Etiqueta <select>

Genera una lista desplegable. Por defecto viene vacía, para definir las opciones necesitamos usar otra etiqueta. También se llama combobox.

Etiqueta <option>

Define una opción adentro de una lista desplegable. Es importante que siempre esté dentro de la etiqueta <select>.

Etiqueta <button>

Sirve para generar un botón que nos permita enviar la información de nuestro formulario.
