Modelo de caja

Es la manera en que se comportan los elementos de html. Todos las etiquetas de html tienen un comportamiento default:

  • elemento de bloque
  • elemento de línea

El box-model busca implementar propiedades para manipular:

  1. Tamaño (ancho y alto)
  2. Relleno
  3. Borde
  4. Márgen

Elementos de bloque

Son aquellos que ocupan todo el espacio del que disponen a lo ancho. Un elemento de bloque, recibe facilmente todas las propiedades del box-model. Elementos de bloque más comunes:

  • h1, h2... h6
  • p
  • div
  • ul, ol, li
  • nav, header

Elementos de línea

Son aquellos que ocupan solamente la extensión de su contenido a lo ancho. Un elemento de línea, no recibe muy bien las propiedas del box-model. Elementos de línea más comunes:

  • a
  • span
  • b, strong
  • i, em

Propiedades del box-model

width (ancho)

La propiedad width, nos permite aplicar tamaño a lo ancho a un elemento que tome esta propiedad. El valor que definamos para el width, puede estar expresado en: pixeles (px) ó porcentaje (%). Ejemplo:

width: 300px; ó width: 45%;

height (alto)

La propiedad height, nos permite aplicar tamaño a lo alto a un elemento que tome esta propiedad. El valor que definamos para el height, puede estar expresado en: pixeles (px) ó porcentaje (%). Sin embargo siempre va a convenir aplicar un tamaño en px. Ó quizás convenga NO aplicar directamente esta propiedad. Ejemplo:

height: 120px;

La propiedad padding puede recibir 1, 2, 3 ó 4 valores, expresados en pixeles.

padding con 1 valor:

padding: 30px;
/* se aplica por igual a los 4 costados (top, right, bottom, left)*/

padding con 2 valores:

padding: 10px 5px;
/* 1er valor (top y bottom) 2do valor (right y left) */

padding con 3 valores:

padding: 40px 25px 10px;
/* 1er valor (top) 2do valor (right y left) 3er valor (bottom) */

padding con 4 valores:

padding: 40px 25px 10px 0;
/* 1er valor (top) 2do valor (right) 3er valor (bottom) 4to valor (left) */

La propiedad margin recibe valores expresados en px o %, si aplicamos el valor auto el elemento quedará centrado de manera horizontal si y solo si es un elemento de bloque. El margin funciona muy parecido al padding, pues recibe 1, 2, 3 y hasta 4 valores.

margin: 100px auto;

Cambiando la naturaleza de un elemento

Cuando querramos cambiar el comportamiento por default de un elemento (de línea a bloque) podemos usar la propiedas display. Dentro de los valores que más nos interesan son:

  • block
  • inline
  • inline-block
  • none

Ejemplos:

display: block;
/* lo convierte al elemento en un elemento de bloque */

display: inline;
/* lo convierte al elemento en un elemento de línea */

display: inline-block;
/* lo convierte al elemento en un elemento de semi-bloque */

display: none;
/* Oculta al elemento */
